Form Component APIs

Review props, slots, and visual preview

1. BaseInput

Text input with integrated label and error display. Supports v-model.

Props
modelValue: string — v-model binding
label?: string — renders label above input
id?: string — auto-generated from label if omitted
type?: 'text' | 'email' | 'password' | 'number' | 'tel' | 'url' — default: 'text'
placeholder?: string
required?: boolean
disabled?: boolean
error?: string — red error message below input
minlength?: number
maxlength?: number
Usage
<BaseInput v-model="form.name" label="Name" placeholder="Write your name" required :minlength="2" :maxlength="100" :error="errors.name" /> <BaseInput v-model="form.email" label="Email" type="email" placeholder="email@example.com" required :error="errors.email" />

2. BaseTextarea

Same API as BaseInput but renders a textarea. Adds rows prop.

Props
modelValue: string — v-model binding
label?: string
id?: string
placeholder?: string
required?: boolean
disabled?: boolean
error?: string
rows?: number — default: 4
minlength?: number
maxlength?: number
Usage
<BaseTextarea v-model="form.message" label="Message" placeholder="Write your message here" required :rows="4" :minlength="10" :maxlength="1000" :error="errors.message" />
