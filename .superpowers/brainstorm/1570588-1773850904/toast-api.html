Toast API & Visual Design

Review the composable API and toast appearance

1. useToast() Composable

API
const toast = useToast() // Show a toast toast.success('Message sent!') toast.error('Failed to send message') toast.warning('Connection unstable') toast.info('New version available') // Or with full options toast.show({ message: 'Message sent!', type: 'success', // 'success' | 'error' | 'warning' | 'info' duration: 5000, // ms, 0 = no auto-dismiss }) // Remove a specific toast toast.remove(id) // Clear all toast.clear()
Default durations
success: 5000ms (auto-dismiss)
info: 5000ms (auto-dismiss)
warning: 0 (stays until closed)
error: 0 (stays until closed)
Usage in Contact.vue
const toast = useToast() // Replace inline success div toast.success('Message sent successfully!') // Replace inline error div toast.error(msg)

3. Accessibility

ARIA attributes
Container: role="region" + aria-label="Notifications"
Each toast: role="alert" + aria-live="assertive"
Close button: aria-label="Close notification"
Auto-dismiss: pauses on hover/focus (a11y best practice)
