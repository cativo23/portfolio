1. BaseButton

Polymorphic — renders as <button>, <NuxtLink>, or <a> based on props.

Props
variant: 'primary' | 'secondary' | 'ghost' | 'icon' — default: 'primary'
size: 'sm' | 'md' | 'lg' — default: 'md'
to?: string — renders as NuxtLink
href?: string — renders as <a>
disabled?: boolean
loading?: boolean
type?: 'button' | 'submit' | 'reset' — default: 'button'
Usage Examples
<BaseButton to="/projects">View Projects</BaseButton> <BaseButton variant="primary" type="submit" :loading="sending"> Send Message </BaseButton> <BaseButton variant="ghost" @click="nextPage">Next</BaseButton> <BaseButton variant="icon" href="https://github.com/cativo23"> <LucideGithub /> </BaseButton> <BaseButton variant="secondary" to="/about">Learn More</BaseButton>

2. BaseCard

Content container with Tokyo Night dark styling and optional slots.

Props & Slots
Props:
  hoverable?: boolean — default: true — adds hover:shadow-lg
  padded?: boolean — default: true — adds p-6
Slots:
  #default — main content
  #header — optional top area (image, title bar)
  #footer — optional bottom area (actions, links)
  #badge — positioned top-right overlay
Usage Examples
<BaseCard> <h3>Project Title</h3> <p>Description here</p> </BaseCard> <BaseCard> <template #badge> <BaseBadge color="cyan">Featured</BaseBadge> </template> <h3>My Project</h3> <p>Project description</p> <template #footer> <BaseButton variant="ghost" to="/projects/1"> View Details → </BaseButton> </template> </BaseCard>

4. BaseSectionHeading

Section title with optional DecryptedText animation and subtitle.

Props & Slots
Props:
  title: string
  animated?: boolean — default: false — wraps in DecryptedText
  level?: 2 | 3 — default: 2 — renders h2 or h3
Slots:
  #subtitle — optional text below heading
Usage
<BaseSectionHeading title="Featured Projects" animated> <template #subtitle> Some of my recent work </template> </BaseSectionHeading>
