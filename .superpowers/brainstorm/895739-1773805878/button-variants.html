Which button variants do you need?

Select all that apply. These will be the variant prop options for BaseButton.

A

Primary (Filled)

Solid background, high emphasis. For main CTAs like "View Projects", "Send Message".

B

Secondary (Outlined)

Border only, medium emphasis. For secondary actions alongside a primary button.

C

Ghost (Text only)

No background or border, low emphasis. For pagination, nav items, subtle actions.

D

Icon (Square, icon-only)

Compact square button for icons. For social links, close buttons, toggles.
